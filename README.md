# 我的礼物愿望单

一个移动端优先的礼物愿望单展示网站，支持手动添加/编辑/删除礼物，自动解析商品标题和价格，数据保存在浏览器本地存储。

## 使用方式

直接在手机浏览器打开 GitHub Pages 链接即可使用。

- 管理员密码：`8888`（可在 `app.js` 中修改 `ADMIN_PWD`）
- 添加礼物时：
  - 方式一：粘贴商品链接自动识别（受限于淘宝/京东反爬，成功率较低）
  - 方式二：快速粘贴商品标题和价格，自动填入表单

## 文件结构

- `index.html`：主页面
- `style.css`：样式
- `app.js`：核心逻辑

## 数据说明

礼物数据保存在浏览器 `localStorage` 中，每个访客/每台设备独立存储。
